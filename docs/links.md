# Useful Links

Need to fix this pack or interested in making a pack of your own? These links will help you get started.

1. [PopTracker Repo](https://github.com/black-sliver/PopTracker#readme) - Main repository for PopTracker. Includes some very helpful technical documentation.
2. [PopTracker Package Docs](https://github.com/black-sliver/PopTracker/blob/master/doc/PACKS.md) - Official docs for PopTracker packages.
3. [Pack Creation Guide](https://github.com/black-sliver/PopTracker/blob/master/doc/PackCreation.md) - A pack creation guide created for folks who just want to get started. Written to be friendly to people who don't know coding...but be ye warned, some basic understanding of code will be very helpful in understanding this.
4. [Pack Generation Script](https://github.com/StripesOO7/poptracker-pack-builder) - A script which allows you to generate the shell of a tracker package, including support for Archipelago.
5. [PopTracker Pack Schema](https://poptracker.github.io/schema/packs/) - JSON schemas for poptracker packs.
